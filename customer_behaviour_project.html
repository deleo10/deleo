Project Overview

This project analyzes customer shopping behavior to understand purchasing patterns, spending habits, and customer segmentation. The goal was to provide data-driven insights that help businesses improve marketing strategies and customer retention.

Tools & Technologies

SQL Python (Pandas) Power BI Data Visualization

Key Questions Answered

  • Which products generate the highest revenue?
  • What customer segments spend the most?
  • How does purchase frequency affect revenue?
  • What trends can improve sales performance?

Key Insights

  • Top products generated over 60% of total revenue.
  • Returning customers showed significantly higher average order value.
  • Seasonal trends significantly affected sales.
