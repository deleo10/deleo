Project Overview

This project analyzes Nigeria’s low-budget smartphone market using online retail data (prices in naira), to evaluate pricing strategies, brand performance, hardware specifications, discounts, and value-for-money across phone models

Tools & Technologies

SQL Python (Pandas) Power BI Data Visualization

Key Questions Answered

  • Which brands reduce prices the most?
  • Which RAM–ROM combination dominates the market?
  • Which brand offers the best value?
  • Which Phone Offers the Best deal?

Key Insights

  • A few brands and models drive most of the market value.
  • Mid-range phones offer the best value for money.
  • RAM size strongly influences pricing.
  • High discounts don’t always mean cheaper phones.
